log_n
~~~~~


.. function:: log_n(evm: ethereum.london.vm.Evm, num_topics: ethereum.base_types.U256) -> None
   :noindexentry:

   Appends a log entry, having `num_topics` topics, to the evm logs.

   This will also expand the memory if the data (required by the log entry)
   corresponding to the memory is not accessible.

   :param evm: The current EVM frame.
   :param num_topics: The number of topics to be included in the log entry.
