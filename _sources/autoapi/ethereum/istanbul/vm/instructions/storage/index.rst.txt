sload
~~~~~


.. function:: sload(evm: ethereum.istanbul.vm.Evm) -> None
   :noindexentry:

   Loads to the stack, the value corresponding to a certain key from the
   storage of the current account.

   :param evm: The current EVM frame.

   :raises ~ethereum.istanbul.vm.exceptions.StackUnderflowError: If `len(stack)` is less than `1`.
   :raises ~ethereum.istanbul.vm.exceptions.OutOfGasError: If `evm.gas_left` is less than `50`.

sstore
~~~~~~


.. function:: sstore(evm: ethereum.istanbul.vm.Evm) -> None
   :noindexentry:

   Stores a value at a certain key in the current context's storage.

   :param evm: The current EVM frame.

   :raises ~ethereum.istanbul.vm.exceptions.StackUnderflowError: If `len(stack)` is less than `2`.
   :raises ~ethereum.istanbul.vm.exceptions.OutOfGasError: If `evm.gas_left` is less than `20000`.
